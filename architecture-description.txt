TRAFFIC FLOW:
User → Route 53 → CloudFront → WAF → ALB → Target Group → 
ASG EC2 (1-20 instances) → ElastiCache (if cached) → RDS Aurora

SCALING TRIGGERS:
├── CPU > 70% → Scale UP (+2 instances)
├── CPU < 30% → Scale DOWN (-1 instance)
├── Request count > 10,000/min → Emergency scale to max
└── Response time > 2s → Check cache & scale
